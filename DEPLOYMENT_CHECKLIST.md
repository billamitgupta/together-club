## Backend Deployment (Vercel)

### 3. Vercel Setup
- [ ] Vercel account created
- [ ] Vercel CLI installed (`npm i -g vercel`)
- [ ] Repository connected to Vercel (optional)

### 4. Environment Variables (Vercel)
```bash
# Add these in Vercel Dashboard > Project > Settings > Environment Variables
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/togetherclub
RAZORPAY_KEY_ID=rzp_live_xxxxxxxxxx
RAZORPAY_KEY_SECRET=your_secret_key
NODE_ENV=production
ALLOWED_ORIGINS=https://your-frontend-domain.netlify.app
```

### 5. Deploy Backend
```bash
# From project root
vercel --prod
```

- [ ] Deployment successful


## Troubleshooting Common Issues

### Backend Issues
- **CORS Errors**: Check ALLOWED_ORIGINS environment variable
- **Database Connection**: Verify MongoDB URI and IP whitelist
- **Payment Issues**: Check Razorpay keys and webhook configuration
- **Function Timeouts**: Optimize database queries and API calls

### Frontend Issues
- **Build Failures**: Check Node.js version and dependencies
- **API Connection**: Verify REACT_APP_API_URL points to correct backend
- **Payment Integration**: Ensure Razorpay script loads and keys are correct
- **SEO Issues**: Verify meta tags are being set correctly

### Performance Issues
- **Slow Loading**: Optimize images, enable compression, use CDN
- **High Bounce Rate**: Improve mobile experience and loading speed
- **Low Conversion**: A/B test different CTAs and form designs
